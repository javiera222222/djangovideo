 Simple API Practice Project using Django
 (abajo versión en español)
Description

This is a small project created to practice building REST APIs using Django and Django REST Framework.
It focuses on the fundamentals of creating endpoints, managing serializers, connecting models to a database, and testing basic API functionality.

The project aims to strengthen backend development skills by providing a clean and minimal example of an API built with Django.

Features

CRUD operations (Create, Read, Update, Delete)
RESTful endpoints with Django REST Framework
JSON responses
Basic model and serializer structure
SQLite as default database

Technologies Used

Python 3
Django
Django REST Framework
SQLite

Proyecto simple para practicar el uso de APIs con Django
Descripción

Este es un pequeño proyecto creado para practicar la construcción de APIs REST utilizando Django y Django REST Framework.
Se centra en los fundamentos de la creación de endpoints, el manejo de serializers, la conexión de modelos con una base de datos y la prueba de la funcionalidad básica de una API.

El objetivo del proyecto es fortalecer las habilidades de desarrollo backend mediante un ejemplo limpio y minimalista de una API construida con Django.

Funcionalidades

Operaciones CRUD (Crear, Leer, Actualizar, Eliminar)
Endpoints RESTful con Django REST Framework
Respuestas en formato JSON
Estructura básica de modelos y serializers
SQLite como base de datos por defecto

Tecnologías utilizadas

Python 3
Django
Django REST Framework
SQLite
